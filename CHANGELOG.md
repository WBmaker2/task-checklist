# Changelog

이 프로젝트의 주요 변경 사항을 기록합니다.

## v2.0.18 - 2026-03-16

- 대시보드에 최근 놓친 업무를 바로 처리할 수 있는 보완 추천 섹션을 추가했습니다.

## v2.0.17 - 2026-03-16

- 로컬 JSON 내보내기와 가져오기 백업 기능을 추가했습니다.

## v2.0.16 - 2026-03-16

- 백업·동기화 상태 관리를 AppSyncController로 분리했습니다.

## v2.0.15 - 2026-03-16

- 백업 동기화 스모크 테스트와 공용 sync 유틸을 추가했습니다.

## v2.0.14 - 2026-03-16

- 릴리스 자동화 스크립트와 `CHANGELOG.md` 관리 흐름을 추가했습니다.

## v2.0.13 - 2026-03-16

- 백업 직후 `baseVersion`이 한 단계 뒤처져 자기 자신을 최신 서버로 잘못 감지하던 문제를 수정했습니다.
- `backupUserData()`가 transaction에서 계산한 버전을 우선 사용하도록 바꾸고, 관련 스크립트 캐시 버스터를 갱신했습니다.
