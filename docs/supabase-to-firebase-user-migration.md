# Supabase -> Firebase 사용자 이전 메모

이 문서는 예전 Supabase 백업을 쓰던 사용자가 새 Firebase 백업으로 옮길 때 참고할 간단한 안내문이다.

## 이전 순서

1. 예전 Supabase 버전 앱에서 마지막으로 사용하던 기기를 연다.
2. `백업 복원`을 눌러 최신 데이터가 맞는지 확인한다.
3. 새 Firebase 버전 앱에서 Google 로그인을 완료한다.
4. `지금 백업`을 눌러 현재 데이터를 새 Firebase 백업으로 저장한다.
5. 이후부터는 새 버전 앱만 사용한다.

## 메모

- 예전 앱에서 서버 복원을 한 뒤 새 앱으로 와서 백업하면 된다.
- 이전이 끝난 뒤에는 Firebase 백업만 사용해도 된다.
