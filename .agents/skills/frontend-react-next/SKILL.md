# Frontend React Next

## 목표

- 디자인과 계약 문서를 기반으로 React 또는 Next.js UI를 구현한다.

## 절차

1. `_workspace/01_design_wireframe.md` 와 `_workspace/01_design_component_map.md`를 읽는다.
2. API가 있으면 `_workspace/02_api_contracts.md` 를 읽고 데이터 shape를 맞춘다.
3. 라우트, 페이지, 공통 컴포넌트, 상태 경계를 먼저 정한다.
4. 구현 후 시각 상태와 상호작용 상태를 점검한다.
5. `_workspace/02_frontend_handoff.md` 에 변경 파일, 검증, 남은 리스크를 적는다.

## 기대 출력

- 구현 파일 목록
- 필요한 env 또는 mock 전제
- 미구현 상태가 있으면 정확한 이유

## 주의

- API shape를 임의로 추정하지 않는다.
- 기존 디자인 언어가 있으면 그것을 우선한다.
